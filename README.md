# 📘 Taskboard: 동아리 기반 하루 루틴 플래너 & 집중 타이머

## 🔍 프로젝트 개요

**Taskboard**는 동아리 홈페이지와 개인 맞춤형 루틴 플래너, 집중 타이머(Pomodoro)를 통합한 웹 플랫폼입니다.  
사용자는 로그인 후 대시보드에서 오늘의 일정을 확인하고, 루틴에 따라 집중 타이머를 실행하며 하루를 체계적으로 관리할 수 있습니다.

> 💡 HTML, CSS, JavaScript, Bootstrap, Firebase를 활용하여 구현한 프로젝트입니다.

---

## 🧩 주요 기능

### 🏠 동아리 홈페이지
- 동아리 소개, 활동 안내
- Firebase Authentication 기반 로그인 및 회원가입 기능

### 📋 대시보드
- 오늘의 할 일 목록 표시 (Firestore에서 불러오기)
- 진행률을 보여주는 프로그레스바
- "플래너 보기" 버튼 제공

### 📅 루틴 플래너
- 하루 시간대별 일정 시각화 (타임라인 형태)
- 색상 블록으로 시간대 구분
- 각 항목 클릭 시 타이머 작동

### ⏱️ 집중 타이머 (Pomodoro 방식)
- 25분 집중 / 5분 휴식 주기
- 실시간 타이머 표시 및 애니메이션
- 완료 시 진행률 자동 갱신 및 Firestore 업데이트

### 📈 추가 기능 (선택 사항)
- 차트로 일별/주별 작업 통계 시각화 (Chart.js + Firestore 기반)
- 알림/리마인더 기능 (Firebase Cloud Messaging 연동 가능)
- 모바일 반응형 레이아웃 지원

## ✅ 요구사항 분석

1. 사용자는 Firebase Auth를 통해 회원가입 및 로그인이 가능해야 한다.
2. 로그인한 사용자별로 Firestore 기반의 개인 대시보드를 제공해야 한다.
3. 할 일(Task)을 시간대별로 등록하고 Firestore에 저장할 수 있어야 한다.
4. 일정은 타임라인 형태로 시각화되어야 한다.
5. 각 항목 클릭 시 집중 타이머가 작동해야

## 웹페이지 흐름

[비로그인 사용자]
→ [동아리 홈페이지]
   └ 동아리 사진 + 소개글
   └ 로그인 / 회원가입 버튼

↓

[로그인 후]
→ [대시보드 자동 진입]
   └ 오늘의 할 일 목록 보기
   └ 진행률 확인
   └ 각 할 일을 클릭하면 → 해당 시간 플래너로 이동

↓

[플래너 화면]
   └ 시간대별 일정 블록 시각화
   └ 클릭 시 해당 블록 시간으로 뽀모도로 실행

↓

[집중 타이머]
   └ 집중 시간(25분) 카운트
   └ 완료 시 해당 Task 상태 업데이트
   └ 대시보드로 자동 반영
