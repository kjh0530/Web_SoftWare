#Taskboard: 동아리 기반 하루 루틴 플래너 & 집중 타이머

## 🔍 프로젝트 개요

**Planity**는 동아리 홈페이지와 개인 맞춤형 루틴 플래너, 집중 타이머(Pomodoro)를 통합한 웹 플랫폼입니다.  
사용자는 로그인 후 대시보드에서 오늘의 일정을 확인하고, 루틴에 따라 집중 타이머를 실행하며 하루를 체계적으로 관리할 수 있습니다.

> 💡 HTML, CSS, JavaScript, Bootstrap, 서버 기술을 활용하여 구현하였으며, React를 선택적으로 사용할 수 있습니다.

---

## 🧩 주요 기능

### 🏠 동아리 홈페이지
- 동아리 소개, 활동 안내
- 로그인 및 회원가입 기능

### 📋 대시보드 (마이페이지)
- 오늘의 할 일 목록 표시
- 진행률을 보여주는 프로그레스바
- "플래너 보기" 버튼 제공

### 📅 루틴 플래너
- 하루 시간대별 일정 시각화 (타임라인 형태)
- 색상 블록으로 시간대 구분
- 각 항목 클릭 시 타이머 작동

### ⏱️ 집중 타이머 (Pomodoro 방식)
- 25분 집중 / 5분 휴식 주기
- 실시간 타이머 표시 및 애니메이션
- 완료 시 진행률 자동 갱신

### 📈 추가 기능 (선택 사항)
- 차트로 일별/주별 작업 통계 시각화 (Chart.js)
- 알림/리마인더 기능
- 모바일 반응형 레이아웃 지원

## ✅ 요구사항 분석

1. 사용자는 회원가입 및 로그인이 가능해야 한다.
2. 로그인한 사용자별로 개인 대시보드를 제공해야 한다.
3. 할 일(Task)을 시간대별로 등록할 수 있어야 한다.
4. 일정은 타임라인 형태로 시각화되어야 한다.
5. 항목 클릭 시 집중 타이머가 작동해야 한다.
6. 타이머는 현재 상태(집중/휴식)를 시각적으로 표현해야 한다.
7. 완료한 항목은 진행률에 반영되어야 한다.
8. 데이터는 저장되어 다시 불러올 수 있어야 한다.
9. UI는 반응형이어야 하며 직관적으로 구성되어야 한다.
10. 각 항목은 수정/삭제가 가능해야 한다.
